# sw_prototype
Prototype for SW Concept

## Running the client side application

### Project setup

Make sure you are in the client folder

```
yarn install
```

##### Compiles and hot-reloads for development
```
yarn run serve
```

#### Compiles and minifies for production
```
yarn run build
```

#### Run your tests
```
yarn run test
```

#### Lints and fixes files
```
yarn run lint
```
